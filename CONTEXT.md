# WLOC 定位改写

这个项目修改 Apple Wi-Fi/基站网络定位服务返回的坐标，让目标设备可以使用指定的网络定位结果。它不修改 GPS 硬件定位，也不保证 iOS 的最终融合定位一定采用改写结果。

## Language

**WLOC 定位改写**:
通过代理拦截并修改 Apple Wi-Fi/基站网络定位响应；不修改 GPS，也不保证最终系统定位一定采用该结果。
_Avoid_: iOS 虚拟定位、GPS 修改、全局定位修改

**路线模拟**:
让目标网络定位沿一组有序路线点随时间移动的模式；可暂停、继续、停止或循环。
_Avoid_: 动态 GPS、真实导航

**运动档案**:
为路线模拟提供默认速度与定位精度的预设，例如步行、骑行或驾车；不代表系统已经识别到真实运动活动。
_Avoid_: 运动检测、运动传感器伪造

**路线状态**:
路线模拟当前所处的生命周期状态，包括等待、运行、暂停、停止和完成。
_Avoid_: 定位状态

**只检查模式**:
解析 WLOC 响应并给出不含设备标识的诊断摘要，但保持原响应不变的模式。
_Avoid_: 调试定位、原始数据导出

**诊断输出渠道**:
诊断摘要的承载位置，可选择 HTTP 响应头、运行日志或两者；不改变摘要的隐私边界。
_Avoid_: 数据导出格式、远程上报
